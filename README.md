# NFT-Minting-Sprint

## 프로젝트 설명

- 지갑 로그인 후 redux-toolkit을 사용하여 전역변수에 지갑 주소값을 저장합니다.
- 원하는 사진과 이름, 설명을 입력 후에 'Make NFT' 버튼을 누릅니다.
- 가스비를 지불하면 로그인한 지갑 주소로 NFT가 만들어집니다.
